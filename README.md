# GitHub Traffic Dashboard

[tommie-jp](https://github.com/tommie-jp) の**公開リポジトリ**の閲覧統計
(views / clones / stars)を横断表示する自動生成ダッシュボードです。

- 公開ページ: <https://tommie-jp.github.io/github-traffic-dashboard/>
- データ元: 各リポジトリの GitHub Traffic 統計を
  [jgehrcke/github-repo-stats](https://github.com/jgehrcke/github-repo-stats)
  で日次アーカイブしたもの(GitHub の 14 日保持制限を超えて蓄積)。
- `index.html` は CSS/JS/データを内包した自己完結ファイル。

> このリポジトリは表示用の静的サイトのみを含みます。生成ロジックとデータ収集は
> 非公開の別リポジトリで管理しており、**公開リポジトリの統計のみ**を掲載しています。
